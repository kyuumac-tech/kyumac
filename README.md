# KyuuMac - Service Macbook & iMac Jogja

Website multi-page untuk KyuuMac service center MacBook & iMac di Yogyakarta.

## 🌐 Struktur Website

### Halaman Utama:
1. **Home** (`/`) - Landing page overview
2. **Layanan** (`/layanan`) - Detail semua service & perbaikan
3. **Jual Beli** (`/jual-beli`) - Marketplace MacBook bekas
4. **Blog** (`/blog`) - Artikel & tips (placeholder untuk SEO)
5. **Tentang Kami** (`/tentang`) - Profile & visi misi
6. **Kontak** (`/kontak`) - Form kontak & info lengkap

## 🎯 SEO Optimization

### Multi-Page Benefits:
- ✅ Unique meta title & description per halaman
- ✅ Target keyword berbeda per halaman
- ✅ Internal linking structure
- ✅ Lebih banyak indexed pages di Google
- ✅ Better user experience & navigation

### SEO Files:
- `public/sitemap.xml` - Sitemap untuk search engine
- `public/robots.txt` - Robot crawler instructions

## 🚀 Development

### Install Dependencies:
```bash
npm install
```

### Run Dev Server:
```bash
npm run dev
```

### Build for Production:
```bash
npm run build
```

### Preview Production Build:
```bash
npm run preview
```
